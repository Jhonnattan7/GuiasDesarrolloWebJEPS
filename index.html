Sobre mí

Soy estudiante apasionado por la tecnología, esperando aprender desarrolo backend. No soy muy bueno en el diseño o cretividad de wbes pero espero este bien. Esta web asume el rol de "portafolio estudiantil" y contiene los elementos solicitados: datos personales, datos generales del curso, guías de ejercicios (con descripción y enlaces) y un análisis FODA sobre mi desempeño.

Datos generales del curso

Curso: Desarrollo Web I

Descripción: Aprender a crear webs desde cero para lograr interfaces responsivas y accesibles usando HTML y CSS; aplicar layout con Flexbox y desplegar en GitHub Pages.

Contenidos:

  • HTML semántico y accesibilidad
  • CSS: Box model, Flexbox, y media queries
