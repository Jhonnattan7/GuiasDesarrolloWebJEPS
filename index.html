Sobre mí

Soy estudiante apasionado por la tecnología, especialmente por el desarrollo backend y el análisis de datos. Me motiva profundamente la idea de crear soluciones que puedan ayudar a los demás y resolver problemas reales.

Tengo experiencia trabajando con Python, SQL, y algo de Java. Actualmente estoy aprendiendo JavaScript para ampliar mis habilidades en desarrollo web. Aunque reconozco que el diseño y la creatividad visual no son mi fuerte, me esfuerzo por crear interfaces funcionales y accesibles.

Este portafolio estudiantil documenta mi progreso en el curso de Desarrollo Web I, incluyendo datos personales, información del curso, las guías prácticas realizadas durante el ciclo, y un análisis FODA sobre mi desempeño académico.

Datos generales del curso

Curso: Desarrollo Web I

Descripción: Este curso es el primero en la secuencia de desarrollo web y cubre los fundamentos básicos de los lenguajes HTML, CSS y JavaScript, que son los elementos esenciales de cualquier aplicación web. A través del dominio de los elementos del lenguaje HTML y el uso de los elementos de estilo en lenguaje CSS los estudiantes podrán diseñar sitios web atractivos y funcionales. Mediante el uso de los conceptos básicos y sintaxis de JavaScript, los estudiantes podrán crear aplicaciones interactivas de front-end e introducir contenido dinámico. Asimismo, serán capaces de comprender la arquitectura de las aplicaciones front-end, el rol de cada tecnología en la creación de aplicaciones modulares y conocerán las mejores prácticas en el diseño de estas. Finalmente, se discutirán algunas prácticas desventajosas en el desarrollo de software.

FODA — Análisis de Desempeño Académico

Fortalezas

Habilidades técnicas: Comprensión sólida de HTML semántico, manejo eficiente de Flexbox para layouts adaptativos, y dominio mejorado de selectores CSS. He aprendido a integrar Firebase para almacenamiento en tiempo real, lo cual considero una herramienta valiosa.

Metodología de trabajo: Mi mayor fortaleza es no complicarme al crear soluciones. Tengo la habilidad de planificar mis proyectos, establecer objetivos claros y cumplirlos de manera efectiva sin sobre-ingeniería.

Enfoque backend: Tengo bases sólidas en Python y SQL que complementan mi aprendizaje de desarrollo web.

Oportunidades

Tecnologías emergentes: Tengo gran interés en profundizar en bases de datos no relacionales (MongoDB, Firebase), comprender el funcionamiento de servidores y explorar tecnologías de orquestación como Kubernetes.

Desarrollo de APIs: Quiero aprender más sobre cómo funcionan las APIs RESTful y GraphQL, y cómo diseñar arquitecturas backend escalables.

Proyecto personal: Desarrollar una aplicación que ayude a estudiantes universitarios a gestionar su carga académica, tomar decisiones sobre qué hacer si no aprueban una materia, y optimizar su planificación de estudios. Este proyecto combinaría mis conocimientos de backend y análisis de datos.

Áreas de especialización: Explorar más el análisis de datos y desarrollo backend para convertirme en un desarrollador full-stack enfocado en soluciones de alto impacto.

Debilidades

Diseño visual: Mi gusto estético tiende a ser anticuado, a veces uso combinaciones de colores muy oscuros que no son atractivos visualmente. Necesito desarrollar mejor criterio en diseño UI (User Interface).

Experiencia de usuario: Aunque entiendo la funcionalidad, me falta mejorar en principios de UX para crear interfaces más intuitivas y centradas en el usuario.

Documentación: Reconozco que no soy muy bueno documentando mi código y proyectos. Necesito implementar mejores prácticas de documentación técnica para facilitar el mantenimiento y colaboración.

CSS avanzado: Aún necesito profundizar en CSS Grid avanzado y técnicas modernas de animación.

Testing y accesibilidad: Área pendiente que debo desarrollar para crear aplicaciones más robustas y accesibles.

Amenazas

Gestión del tiempo: Las múltiples responsabilidades académicas y personales limitan el tiempo disponible para profundizar en tecnologías y proyectos personales.

Velocidad de cambio tecnológico: El ecosistema de desarrollo web evoluciona rápidamente con nuevos frameworks y herramientas, lo que requiere aprendizaje constante.

Brecha entre frontend y backend: Mientras me enfoco en backend, existe el riesgo de quedarme rezagado en tendencias modernas de frontend que son cada vez más importantes.

Competencia laboral: El mercado exige desarrolladores con conocimientos tanto de diseño como de programación, y mi debilidad en diseño podría ser una desventaja competitiva.
