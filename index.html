Datos generales del curso

Curso: Desarrollo Web I

Descripción: Este curso es el primero en la secuencia de desarrollo web y cubre los fundamentos básicos de los lenguajes HTML, CSS y JavaScript, que son los elementos esenciales de cualquier aplicación web. A través del dominio de los elementos del lenguaje HTML y el uso de los elementos de estilo en lenguaje CSS los estudiantes podrán diseñar sitios web atractivos y funcionales. Mediante el uso de los conceptos básicos y sintaxis de JavaScript, los estudiantes podrán crear aplicaciones interactivas de front-end e introducir contenido dinámico. Asimismo, serán capaces de comprender la arquitectura de las aplicaciones front-end, el rol de cada tecnología en la creación de aplicaciones modulares y conocerán las mejores prácticas en el diseño de estas. Finalmente, se discutirán algunas prácticas desventajosas en el desarrollo de software.

Temario del Curso y Lecturas

UNIDAD I: INTRODUCCIÓN AL HTML

  • Elementos HTML
  • Semántica del HTML
  • Atributos HTML

UNIDAD II: INTRODUCCIÓN A LAS HOJAS DE ESTILO EN CASCADA (CSS)

  • Selectores CSS
  • Propiedades CSS
  • Modelo de caja

UNIDAD III: MAQUETACIÓN DE SITIOS WEB

  • Posicionamiento y visualización
  • Diseño de layout básico
  • Diseño responsivo y media queries
  • Introducción a Flexbox
  • CSS Grid

UNIDAD IV: FRAMEWORKS DE DISEÑO WEB RESPONSIVO

  • Malla de Bootstrap
  • Utilidades para Bootstrap
  • Elementos Bootstrap

UNIDAD V: INTRODUCCIÓN A JAVASCRIPT

  • Introducción a JavaScript y su sintaxis
  • Estructuras de control
  • Arreglos y objetos
  • Funciones y eventos
  • Programación Orientada a Objetos en JavaScript

UNIDAD VI: DOM Y JAVASCRIPT ASÍNCRONO

  • Manipulación del DOM
  • Notación de Objetos de JavaScript (JSON)
  • Almacenamiento local: localStorage y sessionStorage
  • Programación asíncrona en JavaScript
  • AJAX y consumo de APIs
  • Almacenamiento de datos en Firebase
