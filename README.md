# GuiasDesarrolloWebJEPS
Proyectos y tareas de Desarrollo Web I (2025).

Descripción
-----------
Este repositorio contiene las guías, ejercicios y prácticas realizadas durante la asignatura "Desarrollo Web I". Los materiales están organizados por carpetas (por guía o por semana) y contienen ejemplos HTML, CSS y, en algunos casos, JS e imágenes.

Estado
------
Todas las guías han sido completadas y actualizadas. El contenido refleja los temas y prácticas desarrolladas a lo largo del ciclo. Se actualizó el análisis FODA con mayor detalle al finalizar el ciclo.

Estructura principal (resumen)
-------------------------------
- `index.html` — Portafolio estudiantil con información personal, curso, guías y FODA
- `20245184_Guia1/` — Fundamentos de HTML y CSS (`css/`, `img/`)
- `20245184_Guia2/` — Estructura y multimedia (`css/`, `img/`)
- `20245184_Guia3/` — Posicionamiento y CSS avanzado (`css/`, `images/`)
- `20245184_Guia4/` — Responsive design y media queries (`css/`, `images/`)
- `Guia5_20245184/` — Introducción a JavaScript (`js/`, `pages/`, `css/`)
- `Guia6_20245184/` — Funciones, formularios y Bootstrap (`js/`, `pages/`, `bootstrap/`, `css/`)
- `Guia7_20245184/` — Manipulación del DOM con JavaScript (`js/`, `pages/`, `bootstrap/`, `css/`)

Cómo ver el proyecto localmente
------------------------------
- Opción sencilla: abrir los archivos `*.html` directamente con un navegador (doble clic sobre el archivo).
